# EYOC logo PNG collection

This ZIP contains the EYOC logo/image assets that were publicly fetchable as files in this environment.

## Included PNG files

- `png/eyoc-2014.png` — official EYOC 2014 PNG logo.
- `png/eyoc-2015.png` — converted from available EYOC 2015 JPG poster/logo source.
- `png/eyoc-2016.png` — converted from available EYOC 2016 JPG poster/logo source.
- `png/eyoc-2017.png` — converted from available EYOC 2017 JPG logo source.
- `png/eyoc-2022.png` — available EYOC 2022 PNG logo source.
- `png/eyoc-2025.png` — official EYOC 2025 PNG logo.

## Not included / fetch issues

- 2024: official PNG URLs were verified visually, but direct file download failed from this runtime.
- 2026: official Wix image URLs were verified visually, but the server delivered AVIF despite PNG-looking URLs, and the download tool rejected the content-type.
- 2002-2013, 2018-2019, 2021, 2023: no clean directly downloadable public logo PNG was verified in the earlier inventory pass.
- 2020: no separate EYOC edition; postponed to 2022.

See `eyoc_logo_inventory_2002_2026.md` in this ZIP for source details.
