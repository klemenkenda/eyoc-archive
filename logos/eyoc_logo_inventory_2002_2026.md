# EYOC logo inventory, 2002-2026

Purpose: inventory of public logo/branding assets for European Youth Orienteering Championships (EYOC), with preference for PNG.  
Date checked: 2026-06-28.

## Interpretation

- **Direct PNG** = the linked asset itself is PNG and can be saved as-is.
- **PNG candidate / image-search** = web image result shows a logo or poster, but I could not verify a clean direct PNG asset URL.
- **JPG / poster** = usable as source material, but should be converted/cropped to PNG if a uniform archive is needed.
- **PDF / page only** = likely logo appears in a bulletin/site page, but needs extraction or screenshot/cropping.
- **Not found** = I did not find a credible public logo asset in this pass.

## Conversion notes

For JPG/PDF-derived assets, normalize to PNG with ImageMagick after manual crop:

```bash
magick input.jpg -trim +repage eyoc-YYYY.png
```

For PDFs, first export pages at high resolution, then crop:

```bash
magick -density 300 bulletin.pdf[0] page-1.png
magick page-1.png -crop WIDTHxHEIGHT+X+Y +repage eyoc-YYYY.png
```
